# CopyGPT

CopyGPT is a role for ChatGPT that rewrites any headline or combination of words in a variety of copywriting frameworks.

#### Prompt

> Prompt goes here.

### Questions to ask:

1. Can you re-write this headline, "headline here"
2. Can you tell me best framework for this text, "text here"
